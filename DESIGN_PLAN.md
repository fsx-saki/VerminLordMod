# 蛊真人 × Terraria：改编融合计划（总索引）

> 版本：v3.1 | 日期：2026-05-28
> 核心节奏：**1-2转=肉前 | 3-6转=肉后~月总 | 升仙后=模组原创**

---

## 三段式总纲

```
第一段：凡人蛊师（肉前）      一转~二转  ≈ 30~40小时
  → 青茅山篇，蛊师入门的艰辛与家族政治

第二段：蛊师巅峰（肉后~月总）  三转~六转  ≈ 60~80小时
  → 南疆+义天山+北原+天庭，小说最丰富的内容

第三段：蛊仙之路（升仙后原创）  七转~九转+ ≈ 无上限
  → 仙窍建设+道主争夺+尊者博弈+永生追求
```

## 阶段过渡总览

每个阶段的详细过渡机制请查看各阶段计划文件中的"阶段过渡"章节。以下是跨阶段过渡的概要：

| 过渡 | 触发条件 | 世界变化 | 关键继承 |
|------|---------|---------|---------|
| 1→2 | 一转巅峰+贾金生事件完成 | 三寨大比公告，外族NPC到达 | 方源好感度、资质、声望 |
| 2→3 | 二转巅峰+击败地脉守护者 | Hardmode激活，南疆区域解锁 | 血祭路线选择、白凝冰状态 |
| 3→4 | 四转初期+三王传承+散修主线 | 义天山区域生成 | 三王印记、春秋蝉、黑土身份 |
| 4→5 | 五转初期+义天山事件完成 | 北原冰原区域解锁 | 阵营选择、真传碎片、战功 |
| 5→6 | 五转巅峰+天庭前奏+太白云生事件 | 天庭浮岛出现 | 阵营选择、仙僵状态、太白云生存活 |
| 6→7 | 升仙完成+龙公击败+宿命碎裂 | 仙窍激活，太古荒兽刷新 | 全部蛊虫装备、真元→仙元转换 |
| 7→8 | 七转巅峰+仙窍≥3+魂劫通过 | 道痕共鸣事件 | 道系真传 |
| 8→9 | 八转巅峰+道主+2尊者击败 | 大道共鸣事件 | 道主地位 |
| 9→终局 | 九转巅峰+5尊者+方源击败 | 天意异变 | 至尊仙胎蛊 |
