### 1.2 开窍仪式详细设计

**触发条件**：
- 玩家首次进入GuYueCompoundBiome区域
- StoryPhase == NotEntered
- 世界时间在白天（6:00-18:00）

**仪式流程（不可跳过，约3分钟）**：

| 阶段 | 时间 | 内容 | 技术实现 |
|------|------|------|----------|
| 集合 | 0-30s | 学堂家老召集所有新人在广场 | NPC移动到指定位置，播放对话 |
| 致辞 | 30-60s | 学堂家老讲解开窍意义 | 对话框自动推进，不可跳过 |
| 入水 | 60-90s | 玩家走入地下暗河 | 强制移动玩家到暗河位置 |
| 希望蛊聚集 | 90-120s | 白色粒子特效，希望蛊围绕玩家 | DustSystem + 自定义Projectile |
| 资质检测 | 120-150s | 光柱冲天，颜色渐变 | 自定义Draw + ScreenShader |
| 结果公布 | 150-180s | 学堂家老宣布结果，分配蛊虫 | 对话 + 物品给予 |

**希望蛊特效规格**：
- 粒子数量：50-100个白色光点
- 运动模式：从水面螺旋上升，逐渐向玩家聚拢
- 颜色：白色→淡金色（随资质提升颜色越亮）
- 音效：柔和的风铃声

**资质检测光柱规格**：
- 丁等：暗淡白光，高度20格
- 丙等：淡蓝光，高度40格
- 乙等：蓝色光，高度60格
- 甲等：金色光，高度80格
- 十绝体：七彩光柱，高度120格，全屏震动

### 1.3 资质系统（AptitudeSystem）

**新建文件**：`Common/Players/AptitudePlayer.cs`

```csharp
// 核心数据结构
public enum AptitudeGrade
{
    Ding = 0,   // 丁等：2-3成
    Bing = 1,   // 丙等：4-5成
    Yi = 2,     // 乙等：6-7成
    Jia = 3,    // 甲等：8-9成
    ShiJue = 4  // 十绝体：10成
}

public class AptitudePlayer : ModPlayer
{
    public AptitudeGrade Grade;
    public float AptitudePercent; // 2.0 ~ 10.0
    public bool HasAwakened;      // 是否已完成开窍
    public int YuanStonesPerMonth; // 每月元石津贴

    // 资质影响：
    // 1. 真元修炼速度 = 基础速度 × (AptitudePercent / 5.0)
    // 2. 蛊虫炼化成功率 = 基础率 × (AptitudePercent / 5.0)
    // 3. 空窍容量加成 = (int)(AptitudePercent - 2)
    // 4. 每月元石津贴：
    //    十绝体=100, 甲等=40, 乙等=20, 丙等=5, 丁等=1
}
```

**资质随机算法**：
- 基础概率：丁等39.5%, 丙等40%, 乙等15%, 甲等5%, 十绝体0.5%
- 可通过世界种子影响（特定种子保证甲等以上）
- 多人模式：每个玩家独立随机

### 2.2 修炼系统

**青铜真元修炼**：

- 修炼地点：修炼蒲团（CultivationPlatformTile）、打坐台
- 修炼速度：基础1点/秒 × 资质倍率
- 真元上限：一转 = 1000点
- 突破条件：真元满 + 消耗5元石 + 通过心魔镜考验

**心魔镜考验（MiniGame）**：
- 玩家进入心魔镜Tile后触发
- 场景：黑暗空间，出现幻影敌人
- 机制：玩家需在不攻击的情况下闪避30秒
- 失败：真元减少200点，可重试
- 成功：突破成功，修为提升

## 4. 花酒行者传承

### 4.1 触发条件

- 完成贾金生之死事件链（任意路线）
- 玩家修为 >= 一转中阶
- 在青茅山地下洞穴发现隐藏入口
- 随机触发（10%概率进入特定区域时）

### 4.2 地下洞穴设计

**洞穴结构**：

```
入口（隐藏，需SearchSystem发现）
    │
    ▼
[第一层：酒香走廊]
    │   - 环境效果：粉色雾气，移动速度-10%
    │   - 敌人：醉蜂×5（新敌人，飞行型，攻击附带混乱）
    │   - 谜题：按正确顺序点燃4盏酒灯
    │         提示：墙壁上的壁画（饮酒→醉→醒→悟）
    │
    ▼
[第二层：花酒幻境]
    │   - 环境效果：全屏粉色滤镜，幻觉NPC出现
    │   - 幻觉NPC会给出真假混杂的线索
    │   - 谜题：在3个幻影中找到真实的花酒行者
    │         真实特征：脚下的影子是倒置的
    │
    ▼
[第三层：传承之室]
    │   - Boss战：花酒行者残影（一转巅峰实力）
    │   - 战斗机制：
    │     Phase 1(100%-50%)：酒雾攻击，范围减速
    │     Phase 2(50%-0%)：醉拳连击，高伤害近战
    │   - 战斗对话：
    │     "你若能接我三招，传承便是你的。"
    │     "好！有几分胆色！"
    │     "记住，酒能成事，亦能败事……"
    │
    ▼
[传承宝箱]
    ├─ 酒虫(JiuChong)：一转蛊虫，消耗元石恢复生命
    ├─ 千里地狼蛛：一转蛊虫，召唤地狼蛛宠物
    ├─ 花酒行者手札： lore物品，记录花酒行者生平
```
