## 1. 仙窍建设系统

### 1.1 仙窍入口与进入

**已有基础**：`Common/SubWorlds/ImmortalAperture.cs`（Stage6创建）

**仙窍入口物品**：`Content/Items/Special/ImmortalApertureKey.cs`

| 属性 | 值 |
|------|-----|
| 类型 | 消耗品（可重复使用） |
| 使用效果 | 传送至仙窍子世界 |
| 冷却 | 5秒 |
| 限制 | 仅升仙后可用 |

**仙窍进入流程**：
```
使用仙窍入口 → 播放传送动画（1秒）
    ↓
玩家传送至仙窍子世界
    ↓
仙窍内部渲染（深色背景+星光粒子）
    ↓
玩家可在仙窍内自由移动和建设
    ↓
使用"返回"物品 → 传送回主世界
```

### 1.4 仙窍入侵防御

**入侵触发条件**：
- 升仙后，每10个游戏日有5%概率被其他蛊仙入侵
- 入侵者为随机生成的蛊仙NPC（七转~八转）
- 入侵者会破坏仙窍内的Tile和偷取蛊虫

**入侵流程**：
```
入侵触发 → 世界消息："有蛊仙正在入侵你的仙窍！"
    ↓
玩家可选择：
  A. 立即返回仙窍防御
  B. 继续在主世界活动（仙窍自动防御）
    ↓
选择A：传送至仙窍，与入侵者战斗
选择B：阵法自动防御（效率取决于阵法等级）
    ↓
防御成功：入侵者被击退，获得战利品
防御失败：部分Tile被破坏，可能丢失蛊虫
```

**入侵者NPC**：

| 散修蛊仙 | 七转 | 20000 | 120 | 仙元石×5~10 |
| 邪道蛊仙 | 七转 | 25000 | 140 | 仙元石×10~20 + 仙材 |
| 贪婪蛊仙 | 八转 | 50000 | 180 | 仙元石×20~50 + 稀有仙材 |

**随机模式实现**：
```csharp
private DaoType currentChaosMode;
private int modeTimer;
private const int MODE_DURATION = 900; // 15秒

public override void AI()
{
    modeTimer++;
    if (modeTimer >= MODE_DURATION)
    {
        modeTimer = 0;
        // 从50种道系中随机选择
        currentChaosMode = (DaoType)Main.rand.Next(0, 50);
        OnModeChange(currentChaosMode);
    }

    // 根据当前模式执行对应攻击
    ExecuteChaosAttack(currentChaosMode);
}
```

**道主概念**：道主是某个道系的最高掌控者，拥有该道系的终极力量。八转蛊仙可以挑战现有道主，夺取道主地位。

```csharp
public class DaoLordSystem : ModSystem
{
    public static Dictionary<DaoType, DaoLordInfo> DaoLords = new();

    public static bool IsDaoLord(int playerID, DaoType daoType)
    {
        return DaoLords.TryGetValue(daoType, out var info)
            && info.LordPlayerID == playerID;
    }

    public static int GetDaoLordCount(int playerID)
    {
        int count = 0;
        foreach (var kvp in DaoLords)
        {
            if (kvp.Value.LordPlayerID == playerID) count++;
        }
        return count;
    }
}

public class DaoLordInfo
{
    public DaoType DaoType;
    public int LordPlayerID;         // -1表示NPC道主
    public string LordNPCType;       // NPC道主类型
    public int DaoLordLevel;         // 道主等级（1~10）
    public float DaoMastery;         // 道系掌控度（0~100%）
    public List<DaoLordAbility> Abilities; // 道主能力
}
```

```csharp
private Vector2 PredictPlayerPosition(Player player, int framesAhead)
{
    // 基于玩家当前速度和方向预测未来位置
    Vector2 predicted = player.Center + player.velocity * framesAhead;
    // 考虑玩家加速度
    predicted += (player.velocity - player.oldVelocity) * framesAhead * 0.5f;

private void EvadeAttack(Projectile proj)
```
