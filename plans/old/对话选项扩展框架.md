# 对话选项扩展框架设计

## 一、背景设定

本Mod基于《蛊真人》小说世界观。对话选项应围绕蛊修世界的核心玩法展开：
- **蛊虫**：核心战斗/辅助单位
- **真元/元石**：修炼资源与货币
- **修为**：一转~九转的修炼等级
- **家族势力**：古月、白家、熊家等
- **信念系统**：RiskThreshold/ConfidenceLevel/EstimatedPower
- **声望系统**：敌对~盟友
- **搜索系统**：搜打撤玩法

## 二、现有对话选项类型

| 类型 | 颜色 | 用途 |
|------|------|------|
| Normal | 白色 | 普通对话推进 |
| Informative | 蓝色 | 获取情报 |
| Risky | 红色 | 冒险/挑衅/战斗 |
| Trade | 金色 | 交易/购买 |
| Special | 紫色 | 特殊功能 |
| Exit | 灰色 | 退出对话 |

## 三、需要新增的对话选项类型

### 3.1 新增 `DialogueOptionType` 枚举值

```csharp
public enum DialogueOptionType
{
    Normal,       // 普通（白色）
    Informative,  // 情报（蓝色）
    Risky,        // 冒险（红色）
    Trade,        // 交易（金色）
    Special,      // 特殊（紫色）
    Exit,         // 退出（灰色）
    // === 新增 ===
    Combat,       // 战斗（深红） - 直接触发战斗
    Barter,       // 议价（橙色） - 讨价还价
    Quest,        // 任务（青色） - 接受/提交任务
    Social,       // 社交（粉色） - 送礼/结交/结盟
    Craft,        // 炼制（绿色） - 炼制蛊虫/丹药
    Teach,        // 教学（亮蓝） - 传授功法/知识
    Steal,        // 偷窃（暗紫） - 偷窃/抢夺
    Deceive,      // 欺骗（暗红） - 欺骗/伪装
    Ally,         // 结盟（亮金） - 家族结盟
    Betray,       // 背叛（暗金） - 背叛/背刺
}
```

### 3.2 新增效果类型

```csharp
// 1. 交易效果 - 用元石购买物品
public class BuyItemEffect : DialogueEffect
{
    public int ItemType;
    public int Price;       // 元石价格
    public int MaxStack = 1;
}

// 2. 出售效果 - 出售物品获得元石
public class SellItemEffect : DialogueEffect
{
    public int ItemType;
    public int Price;
}

// 3. 炼制效果 - 消耗材料炼制蛊虫
public class RefineGuEffect : DialogueEffect
{
    public int ResultGuType;     // 炼制出的蛊虫类型
    public Dictionary<int, int> Materials;  // 所需材料
    public float SuccessRate;    // 成功率 0-1
}

// 4. 传授效果 - 传授功法/知识
public class TeachEffect : DialogueEffect
{
    public string SkillName;
    public int RequiredLevel;
    public int CostYuanS;
}

// 5. 结盟效果 - 与家族结盟
public class AllyEffect : DialogueEffect
{
    public FactionID Faction;
    public int DurationDays;  // 结盟持续天数
}

// 6. 悬赏效果 - 发布/接受悬赏
public class BountyEffect : DialogueEffect
{
    public int TargetNPCType;
    public int RewardYuanS;
    public string Description;
}

// 7. 治疗效果 - 消耗元石恢复状态
public class HealEffect : DialogueEffect
{
    public int HealAmount;
    public int CostYuanS;
}

// 8. 传送效果 - 传送到特定位置
public class TeleportEffect : DialogueEffect
{
    public string TargetSubworld;  // 目标小世界
    public Vector2 Position;
    public int CostYuanS;
}

// 9. 情报效果 - 揭示地图/标记位置
public class RevealMapEffect : DialogueEffect
{
    public float Radius;  // 揭示范围
    public string RevealLabel;  // 标记名称
}

// 10. 训练效果 - 提升玩家属性
public class TrainEffect : DialogueEffect
{
    public enum TrainType { MaxLife, MaxMana, Defense, Damage }
    public TrainType Type;
    public int Amount;
    public int CostYuanS;
}
```

### 3.3 新增条件类型

```csharp
// 1. 元石数量条件
public class HasYuanSCondition : DialogueCondition

public class QuestCompletedCondition : DialogueCondition
```

## 七、友好对话树重构示例（新增分支）

```
greeting (根节点)
├── 我想买东西 (Trade) → shop → 打开商店
├── 我需要治疗 (Special) → heal → 消耗元石恢复
├── 请教修炼 (Teach) → teach_menu
│   ├── 突破修为 (Teach) → 消耗元石+材料
│   ├── 炼制蛊虫 (Craft) → 消耗材料
│   └── 返回
├── 打听情报 (Informative) → info_menu
│   ├── 附近危险 (Informative) → 揭示地图
│   ├── 家族情报 (Informative) → 获得情报
│   ├── 宝藏位置 (Informative) → 标记宝藏
│   └── 返回
├── 社交互动 (Social) → social_menu
│   ├── 送礼 (Social) → 提升声望
│   ├── 结交 (Social) → 声望+10
│   ├── 结盟 (Ally) → 正式结盟
│   └── 返回
├── 冒险任务 (Quest) → quest_menu
│   ├── 接受悬赏 (Quest) → 获得悬赏
│   ├── 组队探索 (Quest) → 开启副本
│   └── 返回
├── 展示对话树 (Normal) → 原有演示
├── 调试工具 (Special) → 原有调试
└── 再见 (Exit) → 结束
```
