# 对话树调试方案

## 一、设计目标

信念系统（BeliefState）目前存在一些问题（信念更新逻辑、态度计算等），在修复之前，我们需要一套**独立的调试工具**来展示和测试对话树功能。

调试方案的核心思路：
1. **调试NPC** — 一个友好的城镇NPC，不依赖信念系统，直接使用预设的对话树
2. **调试道具** — 可以设置NPC的信念状态、态度、性格等，用于测试对话树的各种分支
3. **对话树展示** — 通过调试NPC完整展示多级对话树功能

## 二、调试NPC：对话树演示员

### 2.1 NPC 设计

```
名称: 对话树演示员 (DialogueTreeDemoNPC)
类型: 城镇NPC (townNPC = true)
位置: 任意（通过生成道具召唤）
性格: 友善 (Benevolent)
态度: 始终 Friendly（不依赖信念系统）
功能: 展示完整的对话树系统
```

### 2.2 对话树结构

```
[greeting] 欢迎来到对话树演示！
  ├─ [what_is_this] "这是什么？"
  │   └─ 解释对话树概念 → 回到 greeting
  │
  ├─ [show_tree] "展示对话树"
  │   ├─ [tree_basic] "基础对话树"
  │   │   ├─ 单选项 → 回到上级
  │   │   ├─ 多选项 → 回到上级
  │   │   └─ 条件选项（需要道具）→ 回到上级
  │   ├─ [tree_conditional] "条件分支演示"
  │   │   ├─ 信念条件 → 展示不同信念下的不同选项
  │   │   ├─ 声望条件 → 展示不同声望下的不同选项
  │   │   └─ 物品条件 → 展示持有/未持有道具的不同选项
  │   └─ [tree_effect] "对话效果演示"
  │       ├─ 修改信念 → 展示信念变化
  │       ├─ 给予物品 → 获得演示道具
  │       └─ 触发事件 → 展示特殊效果
  │
  ├─ [debug_tools] "调试工具"
  │   ├─ [set_belief] "设置信念状态"
  │   │   ├─ 设置 RiskThreshold = 0.1（自信）
  │   │   ├─ 设置 RiskThreshold = 0.5（中立）
  │   │   ├─ 设置 RiskThreshold = 0.9（恐惧）
  │   │   ├─ 设置 ConfidenceLevel = 0.1（陌生）
  │   │   ├─ 设置 ConfidenceLevel = 0.5（熟悉）
  │   │   └─ 设置 ConfidenceLevel = 0.9（信任）
  │   ├─ [set_attitude] "设置态度"
  │   │   ├─ Friendly → 友好对话树
  │   │   ├─ Wary → 警惕对话树

  │   │   └─ Fearful → 恐惧对话树

      └─ 结束对话
```

### 2.3 NPC 代码设计

```csharp
[AutoloadHead]
public class DialogueTreeDemoNPC : ModNPC
{
    // ===== 调试用信念状态（可被道具修改）=====
    public BeliefState DebugBelief;
    public GuAttitude DebugAttitude = GuAttitude.Friendly;
    public GuPersonality DebugPersonality = GuPersonality.Benevolent;
    
    // ===== 对话树引用 =====
    private DialogueTree _dialogueTree;
    private string _currentNodeID = "greeting";
    
    public override void SetDefaults()
    {
        NPC.width = 18;
        NPC.height = 40;
        NPC.townNPC = true;
        NPC.friendly = true;
        // ... 标准城镇NPC设置
    }
    
    public override string GetChat()
    {
        // 返回当前对话树节点的NPC文本
        return GetCurrentNodeText();
    }
    
    public override void SetChatButtons(ref string button, ref string button2)
    {
        // 根据当前节点的选项数量决定按钮
        var node = GetCurrentNode();
        if (node.Options.Count == 0)
        {
            button = "结束对话";
            button2 = null;
        }
        else if (node.Options.Count == 1)
        {
            button = node.Options[0].Text;
            button2 = null;
        }
        else
        {
            button = node.Options[0].Text;
            button2 = node.Options.Count > 1 ? node.Options[1].Text : null;
        }
    }
    
    public override void OnChatButtonClicked(bool firstButton, ref string shop)
    {
        var node = GetCurrentNode();
        int index = firstButton ? 0 : 1;
        
        if (index < node.Options.Count)
        {
            var option = node.Options[index];
            ApplyEffects(option.Effects);
            _currentNodeID = option.TargetNodeID ?? "greeting";
        }
        
        Main.npcChatText = GetCurrentNodeText();
    }
}
```

### 3.3 信念修改卡（BeliefModifier）

```csharp
// 对准NPC使用，弹出选择菜单
class BeliefModifier : ModItem
{
    public override bool? UseItem(Player player)
    {
        NPC target = GetHoveredNPC();
        if (target == null || !(target.ModNPC is GuMasterBase guMaster))
        {
            Main.NewText("请对准一个蛊师NPC使用。", Color.Yellow);
            return true;
        }
        
        var belief = guMaster.GetBelief(player.name);
        
        // 显示选择菜单（通过原版对话系统）
        Main.npcChatText = $"选择要修改的信念值：\n" +
                          $"当前 RiskThreshold: {belief.RiskThreshold:F2}\n" +
                          $"当前 ConfidenceLevel: {belief.ConfidenceLevel:F2}\n" +
                          $"当前 EstimatedPower: {belief.EstimatedPower:F2}";
        
        // 设置按钮选项
        // button: "降低 RiskThreshold"
        // button2: "提高 RiskThreshold"
        // 后续对话继续选择其他信念值
        
        return true;
    }
}
```

## 四、对话树展示流程

### 4.1 基础展示流程

```
1. 使用"对话树召唤令"生成演示员
2. 右键点击演示员 → 进入对话树
3. 选择"展示对话树" → 看到三种对话树类型
4. 选择"基础对话树" → 体验单选项/多选项/条件选项
5. 选择"条件分支演示" → 体验不同条件下的分支变化
6. 选择"对话效果演示" → 体验信念修改/物品给予/事件触发
```

### 4.2 信念影响展示流程

```
1. 使用"信念调试器"打开面板
2. 选择演示员为目标
3. 设置 RiskThreshold = 0.1（自信）
4. 对话 → 看到自信版本的对话选项
5. 设置 RiskThreshold = 0.9（恐惧）
6. 对话 → 看到恐惧版本的对话选项
7. 对比两种状态下的对话树差异
```

### 4.3 态度影响展示流程

```
1. 使用"态度切换卡"对准演示员
2. 切换为 Wary → 对话 → 看到警惕版本的对话树
3. 切换为 Respectful → 对话 → 看到尊敬版本的对话树
4. 切换为 Hostile → 对话 → 无法对话（CanChat = false）
```

### 4.4 性格影响展示流程

```
1. 使用"性格切换卡"对准演示员
2. 切换为 Aggressive → 对话 → 看到好斗风格的对话
3. 切换为 Greedy → 对话 → 看到贪婪风格的对话（更多交易选项）
4. 切换为 Proud → 对话 → 看到傲慢风格的对话（更多挑衅选项）
```

## 八、调试流程总结

```
┌─────────────────────────────────────────────────────────────┐
│                    对话树调试完整流程                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  步骤 1: 使用"对话树召唤令"                                  │
│  → 在鼠标位置生成对话树演示员 NPC                            │
│                                                              │
│  步骤 2: 右键点击演示员                                      │
│  → 进入对话树，看到 5 个选项的主菜单                         │
│                                                              │
│  步骤 3: 选择"展示对话树"                                    │
│  → 看到三种对话树类型：基础/条件/效果                        │
│                                                              │
│  步骤 4: 体验基础对话树                                      │
│  → 单选项（线性推进）                                        │
│  → 多选项（分支选择）                                        │
│  → 条件选项（需要特定条件）                                  │
│                                                              │
│  步骤 5: 体验对话效果                                        │
│  → 修改信念 → 查看信念变化                                  │
│  → 获得物品 → 背包中出现道具                                │
│  → 触发警告 → 看到消息提示                                  │
│                                                              │
│  步骤 6: 使用调试工具                                        │
│  → 修改 RiskThreshold → 重新对话看变化                      │
│  → 修改态度 → 重新对话看变化                                │
│  → 重置状态 → 回到初始状态                                  │
│                                                              │
│  步骤 7: 使用独立调试道具                                    │
│  → 信念修改卡 → 对准其他蛊师NPC使用                         │
│  → 态度切换卡 → 切换NPC态度                                 │
│  → 性格切换卡 → 切换NPC性格                                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
