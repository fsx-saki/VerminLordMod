## 一、拖尾框架（Content/Trails/）

拖尾框架是项目特效层的核心基础设施，位于 [`Content/Trails/`](Content/Trails/) 目录下，包含 4 个文件：

### 1.1 ITrail 接口 — [`Content/Trails/ITrail.cs`](Content/Trails/ITrail.cs:9)

所有拖尾类型必须实现的接口：

```csharp
public interface ITrail
{
    string Name { get; }
    void Update(Vector2 center, Vector2 velocity);
    void Draw(SpriteBatch sb);
    void Clear();
    bool HasContent { get; }
}
```

### 1.2 TrailManager — [`Content/Trails/TrailManager.cs`](Content/Trails/TrailManager.cs:29)

中央管理器，管理多个 `ITrail` 实例。关键特性：

- **延迟添加/移除**：`Add()` / `Remove()` 操作暂存到 `pendingAdd` / `pendingRemove` 列表，在 `Update()` 时统一处理（`ProcessPending()`），避免遍历时修改集合
- **统一 Additive 混合模式管理**：`Draw()` 自动检测是否需要 Additive 模式（`GhostTrail.UseAdditiveBlend` 或 `LiquidTrail`），自动开关 `SpriteBatch`
- **类型安全查询**：`Get<T>()` / `Has<T>()` 支持按类型查找拖尾实例
- **生命周期**：`Clear()` 清空所有拖尾

### 1.3 GhostTrail — [`Content/Trails/GhostTrail.cs`](Content/Trails/GhostTrail.cs:12)

**虚影拖尾** — 记录历史位置并绘制拉伸贴图，适用于弹幕拖尾、玩家移动拖尾等。

| 配置参数 | 默认值 | 说明 |
|---------|--------|------|
| `MaxPositions` | 16 | 记录位置的最大数量 |
| `RecordInterval` | 2 | 每 N 帧记录一次位置 |
| `TrailTexture` | null | 拖尾贴图 |
| `TrailColor` | (150,220,255) | 拖尾颜色 |
| `Alpha` | 0.8 | 整体透明度 |
| `WidthScale` | 0.4 | 宽度缩放 |
| `LengthScale` | 2.0 | 长度缩放 |
| `UseAdditiveBlend` | true | 是否使用 Additive 混合 |
| `EnableGlow` | false | 是否在头部绘制发光 |
| `GlowColor` | null | 发光颜色（null=TrailColor） |
| `GlowScale` | 1.0 | 发光大小 |
| `Offset` | Vector2.Zero | 固定偏移量 |

**绘制逻辑**：遍历位置数组，每相邻两点绘制一段拉伸贴图，透明度随索引衰减（越旧越透明）。如果 `EnableGlow=true`，在头部绘制 3 层叠加发光。

### 1.4 LiquidTrail — [`Content/Trails/LiquidTrail.cs`](Content/Trails/LiquidTrail.cs:14)

**液态火焰拖尾** — 弹幕经过处产生飞溅碎片，模拟液态火焰效果。

| 配置参数 | 默认值 | 说明 |
|---------|--------|------|
| `MaxFragments` | 40 | 最大碎片数量 |
| `FragmentLife` | 30 | 碎片存活帧数 |
| `SpawnInterval` | 2 | 生成间隔（帧） |
| `SizeMultiplier` | 1.0 | 碎片大小倍率 |
| `TrailTexture` | null | 拖尾贴图 |
| `ColorStart` | (255,80,30) | 碎片初始颜色（年轻） |
| `ColorEnd` | (60,10,20) | 碎片结束颜色（年老） |
| `AirResistance` | 0.92 | 空气阻力系数 |
| `Buoyancy` | 0.08 | 浮力系数（正值向上） |
| `InertiaFactor` | 0.25 | 反向惯性系数 |
| `SplashFactor` | 0.3 | 侧向飞溅系数 |
| `SplashAngle` | 0.8 | 侧向飞溅角度范围 |
| `RandomSpread` | 1.2 | 随机扩散范围 |
| `SpawnOffset` | Vector2.Zero | 生成位置偏移 |

**碎片物理**：每个 Fragment 有 Position、Velocity、Scale、Life、Rotation、RotSpeed。每帧更新：速度×空气阻力 + 浮力 + 位置移动 + 旋转。颜色从 ColorStart 向 ColorEnd 线性插值，大小随生命衰减。

### 1.5 TrailHelper — [`Content/Trails/TrailHelper.cs`](Content/Trails/TrailHelper.cs:11)

**拖尾便捷工厂** — 提供扩展方法快速创建和配置拖尾：

| 方法 | 说明 |
|------|------|
| `AddGhostTrail(manager, tex, color, ...)` | 创建 GhostTrail 并添加到管理器 |
| `AddLiquidTrail(manager, tex, colorStart, colorEnd, ...)` | 创建 LiquidTrail 并添加到管理器 |

| `GetOrAdd<T>(manager)` | 获取或创建指定类型的拖尾 |

## 二、弹幕行为系统（Common/BulletBehaviors/）

弹幕行为系统是特效层的第二层抽象，通过组合模式将特效封装为可复用的 `IBulletBehavior`。

### 2.1 IBulletBehavior 接口 — [`Common/BulletBehaviors/IBulletBehavior.cs`](Common/BulletBehaviors/IBulletBehavior.cs:13)

```csharp
public interface IBulletBehavior
{
    string Name { get; }
    void OnSpawn(Projectile projectile, IEntitySource source);
    void Update(Projectile projectile);
    void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone);
    void OnKill(Projectile projectile, int timeLeft);
    bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch spriteBatch);
}
```

### 2.2 TrailBehavior — [`Common/BulletBehaviors/TrailBehavior.cs`](Common/BulletBehaviors/TrailBehavior.cs:13)

将 `TrailManager` 包装为 `IBulletBehavior`，自动管理拖尾生命周期。

| 属性 | 默认值 | 说明 |
|------|--------|------|
| `TrailManager` | new TrailManager() | 拖尾管理器实例 |
| `AutoDraw` | true | 是否在 PreDraw 中自动绘制拖尾 |
| `SuppressDefaultDraw` | false | 是否阻止引擎默认绘制 |

**生命周期**：
- `OnSpawn`：空操作
- `Update`：调用 `TrailManager.Update(projectile.Center, projectile.velocity)`
- `OnKill`：调用 `TrailManager.Clear()`
- `PreDraw`：如果 `AutoDraw=true` 则绘制拖尾，返回 `!SuppressDefaultDraw`

## 三、自定义 Dust 系统（Content/Dusts/）

### 3.1 概览

[`Content/Dusts/`](Content/Dusts/) 目录下包含 **51 个自定义 ModDust 类**，每个对应一个 Dao 流派。这些只是切换武器后显示在角色头上的，用于表示这是什么道的武器

### 3.2 统一结构

所有 51 个 Dust 类共享完全相同的代码结构（约 30 行）：

```csharp
public class XxxDust : ModDust
{
    public override void OnSpawn(Dust dust) {
        dust.noGravity = true;
        dust.alpha = 100;
        dust.color = Color.White;
        dust.rotation = 0;
        dust.frame = Texture2D.Frame(1, 1, 0, 0, 10, 10);
        dust.scale = 1;
    }
    public override bool Update(Dust dust) {
        dust.position -= Vector2.UnitY;  // 缓慢上浮
        dust.alpha += 5;                 // 逐渐透明
        dust.fadeIn++;
        if (dust.fadeIn > 60 * 2)        // 2秒后消失
            dust.active = false;
        return false;
    }
}
```

**共同特征**：
- `noGravity = true` — 无重力
- `alpha = 100` — 初始半透明
- `color = Color.White` — 白色（可通过 Dust.color 覆盖）
- 缓慢上浮（`position -= Vector2.UnitY`）
- 逐渐透明（`alpha += 5`）
- 2 秒后自动消失

PlayerEffectDrawSystem
