# 弹幕行为抽象重构规划

> **项目**: VerminLordMod（蛊真人Mod）
> **目标**: 抽象弹幕行为类型，为后续 Boss 战提供可复用的弹幕组件
> **日期**: 2026-05-02

---

## 1. 现状分析

### 1.1 现有弹幕实现模式

通过对 `Content/Projectiles/` 下 69 个弹幕文件的抽样分析（阅读了 15+ 个典型文件），发现以下共性模式：

#### 模式 A：直线飞行 + 拖尾（占比约 70%）
代表文件：[`GoldNeedleProj.cs`](Content/Projectiles/GoldNeedleProj.cs)、[`SwordQiProj.cs`](Content/Projectiles/SwordQiProj.cs)、[`DingShenProj.cs`](Content/Projectiles/DingShenProj.cs)、[`LoveToLeaveProj.cs`](Content/Projectiles/LoveToLeaveProj.cs)、[`WaterArrowProj.cs`](Content/Projectiles/WaterArrowProj.cs)、[`IceAwl.cs`](Content/Projectiles/IceAwl.cs)、[`RedNeedle.cs`](Content/Projectiles/RedNeedle.cs)、[`PoisonNeedle.cs`](Content/Projectiles/PoisonNeedle.cs)、[`BloodSkullProj.cs`](Content/Projectiles/BloodSkullProj.cs)

```csharp
class XxxProj : ModProjectile
{
    private readonly TrailManager trailManager = new TrailManager();

    public override void SetDefaults() { /* 标准参数 */ }

    public override void AI() {
        trailManager.Update(Projectile.Center, Projectile.velocity);
        Projectile.rotation = Projectile.velocity.ToRotation() + (float)(0.5 * MathHelper.Pi);
    }

    public override void OnSpawn(IEntitySource source) {
        Projectile.rotation = (float)Math.Atan2(Projectile.velocity.X, -Projectile.velocity.Y);
        // 初始化拖尾（颜色不同，参数几乎相同）
        trailManager.AddGhostTrail(trailTex, color: ..., maxPositions: 16, ...);
    }

    public override bool PreDraw(ref Color lightColor) {
        trailManager.Draw(Main.spriteBatch);
        return true;
    }
}
```

**代码重复量**: 每个文件约 60-70 行，其中约 50 行是完全重复的样板代码。

#### 模式 B：追踪/导向 + 拖尾（占比约 15%）
代表文件：[`CycloneProj.cs`](Content/Projectiles/CycloneProj.cs)、[`PineNeedleProj.cs`](Content/Projectiles/PineNeedleProj.cs)

```csharp
// CycloneProj 的追踪逻辑（硬编码在 AI 中）
Vector2 targetPos = Main.MouseWorld;
var targetVel = Vector2.Normalize(targetPos - Projectile.Center) * 10f;
Projectile.velocity = (targetVel + Projectile.velocity * 10) / 11f;
```

#### 模式 C：特效弹幕 + 发光绘制（占比约 10%）
代表文件：[`MoonlightProj.cs`](Content/Projectiles/MoonlightProj.cs)、[`MeteorProj.cs`](Content/Projectiles/MeteorProj.cs)、[`ShaZhaoJiuYueZhan.cs`](Content/Projectiles/ShaZhaoJiuYueZhan.cs)

这些弹幕在 `PreDraw` 中手动管理 `SpriteBatch` 状态（Additive 混合 → 正常混合），实现发光层效果。这部分代码在每个文件中完全重复。

#### 模式 D：Boss 专用弹幕（占比约 5%）
[`ElectricWolfKing.cs`](Content/NPCs/Boss/ElectricWolfKing.cs) 的 `DoSecondStage_SpawnEyes()` 方法直接使用 `Projectile.NewProjectile` 发射原版弹幕 `ProjectileID.CultistBossLightningOrb`，未使用任何 Mod 自定义弹幕。

### 1.2 发现的痛点

| 痛点 | 描述 | 影响范围 |
|------|------|----------|
| **代码重复** | 每个弹幕文件独立实现 `SetDefaults`、`AI`、`OnSpawn`、`PreDraw`，大量重复 | 全部 69 个弹幕 |
| **行为混合** | 追踪逻辑、旋转逻辑、粒子特效全部硬编码在 `AI()` 中，无法组合复用 | CycloneProj, PineNeedleProj 等 |
| **拖尾初始化重复** | `AddGhostTrail` 调用参数几乎相同，仅颜色不同 | 约 50+ 个弹幕 |
| **PreDraw 样板代码** | 发光弹幕的 `SpriteBatch` 状态切换代码完全一致 | MoonlightProj, MeteorProj, ShaZhaoJiuYueZhan |
| **无行为组合机制** | 无法通过配置组合"追踪 + 旋转 + 拖尾"等行为 | 全部弹幕 |
| **Boss 弹幕发射原始** | 直接使用 `Projectile.NewProjectile` + 原版类型，缺乏 Mod 弹幕复用 | ElectricWolfKing |

### 1.3 与现有系统的耦合

| 系统 | 耦合方式 | 现状 |
|------|----------|------|
| **TrailManager** | 每个弹幕 `new TrailManager()`，在 `AI/PreDraw` 中手动调用 | 良好，接口清晰，可直接复用 |
| **ITrail** | `GhostTrail` / `LiquidTrail` 通过 `TrailManager` 管理 | 良好，可继续使用 |
| **EventBus** | 目前仅用于 `GuWorldEvent`，未与弹幕系统关联 | 零耦合，可扩展 |
| **SmoothMovement** | 位于 `Content/SmoothMovement/`，包含 4 个子系统：Interpolators、Trackers、Orbiters、StateMachine | 已实现，可直接集成 |

## 2. 目标架构设计

### 2.1 核心接口：`IBulletBehavior`

```csharp
/// <summary>
/// 弹幕行为接口 — 所有弹幕行为必须实现此接口
/// </summary>
public interface IBulletBehavior
{
    /// <summary>行为名称（调试/日志用）</summary>
    string Name { get; }

    /// <summary>弹幕生成时调用</summary>
    void OnSpawn(Projectile projectile, IEntitySource source);

    /// <summary>每帧更新（在 BaseBullet.AI() 中遍历调用）</summary>
    void Update(Projectile projectile);

    /// <summary>命中 NPC 时调用</summary>
    void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone);

    /// <summary>弹幕销毁时调用</summary>
    void OnKill(Projectile projectile, int timeLeft);

    /// <summary>绘制（可选，默认不绘制）</summary>
    bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb);
}
```

### 2.2 弹幕基类：`BaseBullet`

```csharp
/// <summary>
/// 弹幕基类 — 所有 Mod 弹幕应继承此类而非直接继承 ModProjectile
/// 内部维护行为列表，在 AI() 中遍历调用
/// </summary>
public abstract class BaseBullet : ModProjectile
{
    protected List<IBulletBehavior> behaviors = new();
    protected TrailManager trailManager = new();

    /// <summary>子类在此方法中注册行为</summary>
    public abstract void ConfigureBehaviors();

    /// <summary>添加行为</summary>
    protected void AddBehavior(IBulletBehavior behavior) => behaviors.Add(behavior);

    /// <summary>移除行为</summary>
    protected void RemoveBehavior<T>() where T : IBulletBehavior
        => behaviors.RemoveAll(b => b is T);

    /// <summary>获取指定类型的行为</summary>
    protected T GetBehavior<T>() where T : class, IBulletBehavior
        => behaviors.FirstOrDefault(b => b is T) as T;

    // ===== 生命周期 =====

    public sealed override void SetDefaults()
    {
        // 设置默认值
        Projectile.aiStyle = -1;
        // 子类可重写 ConfigureDefaults 来定制
        ConfigureDefaults();
    }

    public virtual void ConfigureDefaults() { }

    public sealed override void OnSpawn(IEntitySource source)
    {
        ConfigureBehaviors();
        foreach (var b in behaviors) b.OnSpawn(Projectile, source);
    }

    public sealed override void AI()
    {
        trailManager.Update(Projectile.Center, Projectile.velocity);
        foreach (var b in behaviors) b.Update(Projectile);
    }

    public sealed override void OnHitNPC(NPC target, NPC.HitInfo hit, int damageDone)
    {
        foreach (var b in behaviors) b.OnHitNPC(Projectile, target, hit, damageDone);
    }

    public sealed override void OnKill(int timeLeft)
    {
        foreach (var b in behaviors) b.OnKill(Projectile, timeLeft);
    }

    public sealed override bool PreDraw(ref Color lightColor)
    {
        bool drawn = false;
        foreach (var b in behaviors)
            if (b.PreDraw(Projectile, ref lightColor, Main.spriteBatch))
                drawn = true;
        // 默认绘制拖尾
        if (!drawn) trailManager.Draw(Main.spriteBatch);
        return !drawn; // 如果行为已绘制，返回 false 阻止默认绘制
    }
}
```

#### 2.3.2 `PredictBehavior` — 预判型

```csharp
/// <summary>
/// 预判行为：基于玩家速度与加速度计算提前量
/// 公式：提前位置 = 玩家位置 + 玩家速度 × 飞行时间 + 0.5 × 玩家加速度 × 飞行时间²
/// </summary>
public class PredictBehavior : IBulletBehavior
{
    public string Name => "Predict";
    public float Speed { get; set; } = 10f;
    public float PredictionFactor { get; set; } = 1.0f; // 预判系数，1.0=完全预判

    public void OnSpawn(Projectile projectile, IEntitySource source)
    {
        Player target = Main.player[projectile.owner];
        float distance = Vector2.Distance(projectile.Center, target.Center);
        float flightTime = distance / Speed; // 估算飞行时间

        // 计算预判位置
        Vector2 predictedPos = target.Center
            + target.velocity * flightTime * PredictionFactor
            + target.acceleration * 0.5f * flightTime * flightTime * PredictionFactor;

        Vector2 direction = predictedPos - projectile.Center;
        direction.Normalize();
        projectile.velocity = direction * Speed;
    }

    public void Update(Projectile projectile) { }
    public void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone) { }
    public void OnKill(Projectile projectile, int timeLeft) { }
    public bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb) => false;
}
```

#### 2.3.3 `HomingBehavior` — 追踪型

```csharp
/// <summary>
/// 追踪行为：使用平滑转向追踪最近敌人
/// 可配置追踪强度、最大转向角、锁定距离
/// </summary>
public class HomingBehavior : IBulletBehavior
{
    public string Name => "Homing";
    public float Speed { get; set; } = 10f;
    public float HomingStrength { get; set; } = 0.1f; // 追踪强度（0~1）
    public float MaxTurnAngle { get; set; } = MathHelper.PiOver4; // 最大转向角
    public float MaxLockDistance { get; set; } = 800f; // 最大锁定距离
    public bool TargetPlayers { get; set; } = false; // 是否追踪玩家（用于敌对弹幕）

    private NPC lockedTarget;

    public void OnSpawn(Projectile projectile, IEntitySource source)
    {
        lockedTarget = null;
    }

    public void Update(Projectile projectile)
    {
        // 寻找目标
        if (lockedTarget == null || !lockedTarget.active || Vector2.Distance(projectile.Center, lockedTarget.Center) > MaxLockDistance)
        {
            lockedTarget = FindClosestTarget(projectile);
        }

        if (lockedTarget != null)
        {
            Vector2 desiredDir = lockedTarget.Center - projectile.Center;
            desiredDir.Normalize();
            Vector2 currentDir = projectile.velocity.SafeNormalize(Vector2.Zero);

            // 平滑转向（限制最大转向角）
            float turnAngle = (float)Math.Acos(MathHelper.Clamp(
                Vector2.Dot(currentDir, desiredDir), -1f, 1f));
            float maxTurn = MaxTurnAngle * HomingStrength;
            float actualTurn = Math.Min(turnAngle, maxTurn);

            Vector2 newDir = Vector2.Lerp(currentDir, desiredDir, actualTurn / Math.Max(turnAngle, 0.01f));
            newDir.Normalize();
            projectile.velocity = newDir * Speed;
        }

        projectile.rotation = projectile.velocity.ToRotation() + MathHelper.PiOver2;
    }

    private NPC FindClosestTarget(Projectile projectile)
    {
        NPC closest = null;
        float minDist = MaxLockDistance;
        foreach (var npc in Main.npc)
        {
            if (npc.active && !npc.friendly && npc.CanBeChasedBy())
            {
                float dist = Vector2.Distance(projectile.Center, npc.Center);
                if (dist < minDist) { minDist = dist; closest = npc; }
            }
        }
        return closest;
    }

    public void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone) { }
    public void OnKill(Projectile projectile, int timeLeft) { }
    public bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb) => false;
}
```

#### 2.3.4 `SpreadBehavior` — 铺场型

```csharp
/// <summary>
/// 铺场行为：生成时分裂为多个子弹幕，形成弹幕网
/// </summary>
public class SpreadBehavior : IBulletBehavior
{
    public string Name => "Spread";
    public int BulletCount { get; set; } = 5;
    public float SpreadAngle { get; set; } = MathHelper.PiOver4; // 总扩散角度
    public int BulletType { get; set; } // 子弹幕类型
    public float BulletSpeed { get; set; } = 8f;
    public int BulletDamage { get; set; } = 0; // 0 表示继承母弹幕伤害

    public void OnSpawn(Projectile projectile, IEntitySource source)
    {
        if (BulletType == 0) return; // 未配置子弹幕类型

        float startAngle = -SpreadAngle / 2f;
        float step = SpreadAngle / (BulletCount - 1);

        for (int i = 0; i < BulletCount; i++)
        {
            float angle = startAngle + step * i;
            Vector2 dir = projectile.velocity.SafeNormalize(Vector2.Zero).RotatedBy(angle);
            int damage = BulletDamage > 0 ? BulletDamage : projectile.damage;

            Projectile.NewProjectile(
                source,
                projectile.Center,
                dir * BulletSpeed,
                BulletType,
                damage,
                projectile.knockBack,
                projectile.owner
            );
        }

        // 母弹幕立即消失
        projectile.Kill();
    }

    public void Update(Projectile projectile) { }
    public void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone) { }
    public void OnKill(Projectile projectile, int timeLeft) { }
    public bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb) => false;
}
```

#### 2.3.5 `WaveBehavior` — 波浪型（可选）

```csharp
/// <summary>
/// 波浪行为：弹幕沿正弦波轨迹飞行
/// </summary>
public class WaveBehavior : IBulletBehavior
{
    public string Name => "Wave";
    public float Amplitude { get; set; } = 20f;   // 振幅（像素）
    public float Frequency { get; set; } = 0.1f;  // 频率
    public float Speed { get; set; } = 8f;

    private Vector2 baseDirection;
    private float traveled;

    public void OnSpawn(Projectile projectile, IEntitySource source)
    {
        baseDirection = projectile.velocity.SafeNormalize(Vector2.Zero);
        traveled = 0;
    }

    public void Update(Projectile projectile)
    {
        traveled += projectile.velocity.Length();
        Vector2 perpendicular = baseDirection.RotatedBy(MathHelper.PiOver2);
        float offset = (float)Math.Sin(traveled * Frequency) * Amplitude;

        projectile.velocity = baseDirection * Speed + perpendicular * offset * 0.1f;
        projectile.rotation = projectile.velocity.ToRotation() + MathHelper.PiOver2;
    }

    public void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone) { }
    public void OnKill(Projectile projectile, int timeLeft) { }
    public bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb) => false;
}
```

### 2.4 拖尾行为包装

将现有 [`TrailManager`](Content/Trails/TrailManager.cs) 包装为 `IBulletBehavior` 实现：

```csharp
/// <summary>
/// 拖尾行为 — 将 TrailManager 包装为 IBulletBehavior
/// </summary>
public class TrailBehavior : IBulletBehavior
{
    public string Name => "Trail";
    public TrailManager TrailManager { get; private set; } = new();

    // 便捷方法：添加虚影拖尾
    public void AddGhostTrail(Texture2D texture, Color color, int maxPositions = 16,
        float widthScale = 0.5f, float lengthScale = 1.0f, float alpha = 0.5f,
        int recordInterval = 2, bool enableGlow = false)
    {
        TrailManager.AddGhostTrail(texture, color, maxPositions, widthScale,
            lengthScale, alpha, recordInterval, enableGlow);
    }

    // 便捷方法：添加液态火焰拖尾
    public void AddLiquidTrail(Texture2D texture, Color colorStart, Color colorEnd,
        int maxFragments = 40, int fragmentLife = 30)
    {
        TrailManager.AddLiquidTrail(texture, colorStart, colorEnd, maxFragments, fragmentLife);
    }

    public void OnSpawn(Projectile projectile, IEntitySource source) { }
    public void Update(Projectile projectile) => TrailManager.Update(projectile.Center, projectile.velocity);
    public void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone) { }
    public void OnKill(Projectile projectile, int timeLeft) => TrailManager.Clear();
    public bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb)
    {
        TrailManager.Draw(sb);
        return true; // 已绘制拖尾
    }
}
```

#### `TrackerBehavior` — 将 ITracker 包装为 IBulletBehavior

```csharp
/// <summary>
/// 追踪行为 — 将 ITracker 包装为 IBulletBehavior
/// 支持 DirectTracker / PredictiveTracker / SpiralApproachTracker
/// </summary>
public class TrackerBehavior : IBulletBehavior
{
    public string Name => "Tracker";
    private readonly ITracker tracker;
    private readonly IInterpolator interpolator;
    private NPC lockedTarget;
    public float MaxLockDistance { get; set; } = 800f;

    public TrackerBehavior(ITracker tracker, IInterpolator interpolator = null)
    {
        this.tracker = tracker;
        this.interpolator = interpolator ?? new ExponentialSmoothInterpolator(10f);
    }

    public void OnSpawn(Projectile projectile, IEntitySource source) { }

    public void Update(Projectile projectile)
    {
        // 寻找目标
        if (lockedTarget == null || !lockedTarget.active ||
            Vector2.Distance(projectile.Center, lockedTarget.Center) > MaxLockDistance)
        {
            lockedTarget = FindClosestTarget(projectile);
        }

        if (lockedTarget != null)
        {
            Vector2 targetVel = tracker.GetTargetVelocity(
                projectile.Center, projectile.velocity,
                lockedTarget.Center, lockedTarget.velocity);

            projectile.velocity = interpolator.UpdateVelocity(projectile.velocity, targetVel);
        }

        projectile.rotation = projectile.velocity.ToRotation() + MathHelper.PiOver2;
    }

    private NPC FindClosestTarget(Projectile projectile) { /* ... */ }

    public void OnHitNPC(...) { }
    public void OnKill(...) { }
    public bool PreDraw(...) => false;
}
```

```csharp
/// <summary>
/// 轨道行为 — 将 IOrbiter 包装为 IBulletBehavior
/// 弹幕围绕指定中心点（通常是玩家）做轨道运动
/// </summary>
public class OrbiterBehavior : IBulletBehavior
{
    public string Name => "Orbiter";
    private readonly IOrbiter orbiter;
    private readonly IInterpolator interpolator;
    private int ownerIndex;

    public OrbiterBehavior(IOrbiter orbiter, IInterpolator interpolator = null, int ownerIndex = 0)
    {
        this.orbiter = orbiter;
        this.interpolator = interpolator ?? new LerpSmoothInterpolator(0.08f);
        this.ownerIndex = ownerIndex;
    }

    public void OnSpawn(Projectile projectile, IEntitySource source) { }

    public void Update(Projectile projectile)
    {
        Player owner = Main.player[projectile.owner];
        Vector2 orbitTarget = orbiter.GetOrbitTarget(owner.Center);
        Vector2 targetVel = (orbitTarget - projectile.Center) * 0.15f;
        projectile.velocity = interpolator.UpdateVelocity(projectile.velocity, targetVel);
        projectile.rotation = projectile.velocity.ToRotation() + MathHelper.PiOver2;
    }

    public void OnHitNPC(...) { }
    public void OnKill(...) { }
    public bool PreDraw(...) => false;
}
```

### 5.3 使用示例：星河蛊风格弹幕

```csharp
public class StarRiverBullet : BaseBullet
{
```

### 6.4 兼容性策略

| 策略 | 说明 | 适用场景 |

```csharp
// 适配器：将旧弹幕包装为 IBulletBehavior
public class LegacyBehavior : IBulletBehavior
{
    public string Name => "Legacy";
    private Action<Projectile> aiAction;
    private Action<Projectile, NPC, NPC.HitInfo, int> onHitAction;
    private Action<Projectile, int> onKillAction;

    public LegacyBehavior(Action<Projectile> ai,
        Action<Projectile, NPC, NPC.HitInfo, int> onHit = null,
        Action<Projectile, int> onKill = null)
    {
        aiAction = ai;
        onHitAction = onHit;
        onKillAction = onKill;
    }

    public void OnSpawn(Projectile projectile, IEntitySource source) { }
    public void Update(Projectile projectile) => aiAction?.Invoke(projectile);
    public void OnHitNPC(Projectile projectile, NPC target, NPC.HitInfo hit, int damageDone)
        => onHitAction?.Invoke(projectile, target, hit, damageDone);
    public void OnKill(Projectile projectile, int timeLeft)
        => onKillAction?.Invoke(projectile, timeLeft);
    public bool PreDraw(Projectile projectile, ref Color lightColor, SpriteBatch sb) => false;
}
```

## 10. 总结

本规划提出了一套基于 **组合模式（Composite Pattern）** 的弹幕行为抽象架构，核心思路是：

1. **接口分离**：`IBulletBehavior` 定义弹幕生命周期的各个钩子
2. **基类封装**：`BaseBullet` 封装遍历调用逻辑，子类只需配置行为
3. **行为组合**：通过 `List<IBulletBehavior>` 实现任意行为组合
4. **渐进迁移**：通过 `LegacyBehavior` 适配器实现平滑过渡
5. **数据驱动**（阶段三）：支持通过配置文件定义弹幕行为

该架构可将现有 69 个弹幕的代码量减少约 **60-70%**，同时为后续 Boss 战提供高度可复用的弹幕组件。
