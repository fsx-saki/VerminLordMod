## 1. Hardmode转换

### 1.1 世界变化总览

击败地脉守护者后，世界进入Hardmode。青茅山的封印被打破，深层灵气涌出，南疆大地发生剧变。

**世界变化清单**：

| 变化类别 | 具体内容 | 技术实现 |
|----------|----------|----------|
| 矿脉生成 | 灵晶矿/赤金矿/天晶矿在地下生成 | WorldGen修改 |
| 敌人升级 | 原有敌人获得强化变体 | GlobalNPC修改 |
| 新敌人 | 荒兽级怪物出现 | 新NPC类 |
| 新Biome | 南疆荒野Biome激活 | Biome系统 |
| 天气变化 | 灵气风暴/妖雾天气 | WeatherSystem扩展 |
| NPC迁移 | 部分NPC迁移到南疆 | NPCPersistenceSystem |
| 声望重置 | 南疆势力声望独立计算 | WorldStateMachine扩展 |

### 1.2 新矿石系统

**灵晶矿（LingJingOre）**：
- 生成深度：地下0-50%深度
- 最低采掘力：50%（铜镐级别）
- 熔炼产物：灵晶锭（2矿=1锭）
- 装备等级：相当于Hardmode前中期
- 装备示例：

  - 灵晶甲：35防御，真元恢复+10%
  - 灵晶头饰：12防御，魔法伤害+8%

**赤金矿（ChiJinOre）**：
- 生成深度：地下30-70%深度
- 最低采掘力：65%（银镐级别）
- 熔炼产物：赤金锭（3矿=1锭）
- 装备等级：相当于Hardmode中期
- 装备示例：
  - 赤金刀：80近战伤害，灼烧效果
  - 赤金战甲：45防御，火道伤害+15%

**天晶矿（TianJingOre）**：
- 生成深度：地下60-100%深度
- 最低采掘力：80%（金镐级别）
- 熔炼产物：天晶锭（4矿=1锭）
- 装备等级：相当于Hardmode后期
- 装备示例：
  - 天晶剑：95近战伤害，暴击+15%
  - 天晶法袍：30防御，魔法伤害+20%
  - 天晶护盾：50防御，伤害减免10%

### 3.2 方源（"黑土"）接近剧情

**触发条件**：
- 商心慈关系 >= 2（朋友）
- 完成散修营地主线任务3个以上
- 玩家修为 >= 三转中阶

**剧情流程**：

```
[方源以"黑土"身份出现]
    │   一个自称"黑土"的散修来到营地
    │   外貌：蒙面，低调，但实力深不可测
    │
    ▼
[黑土接近商心慈]
    │   黑土主动与商心慈接触
    │   表面上是寻求合作
    │   实际上方源在利用商心慈的善良
    │
    ▼
[玩家观察/介入]
    │
    ├─ 选择A：警告商心慈
    │   │   "这个人不简单，你要小心。"
    │   │   商心慈："我知道。但每个人都有值得被善待的一面。"
    │   │   黑土好感-10，商心慈好感+5
    │   │
    │   ├─ 选择A1：持续警告 → 商心慈最终听取，与黑土保持距离
    │   │   结果：商心慈安全，但错过黑土提供的资源
    │   │
    │   └─ 选择A2：放弃警告 → 商心慈与黑土合作
    │       结果：商心慈获得资源，但逐渐被利用
    │
    ├─ 选择B：帮助黑土接近商心慈
    │   │   "我可以帮你引荐。"
    │   │   黑土好感+15，商心慈好感-5
    │   │   结果：黑土成功利用商心慈
    │   │
    │   └─ 后续：商心慈被利用后受伤
    │       玩家可选择：
    │       - 安慰商心慈 → 好感+20，获得"慈悲之心"Buff
    │       - 嘲笑商心慈 → 好感-30，商心慈关系线断裂
    │
    └─ 选择C：旁观
        │   不介入，观察事态发展
        │
        └─ 结果取决于后续选择
```

### 3.4 分支结局

**结局A：守护商心慈（正义路线）**
- 条件：持续警告商心慈 + 揭穿黑土身份
- 结果：商心慈安全，黑土离开营地
- 奖励：商心慈关系达到生死之交，获得"慈悲传承"
- "慈悲传承"效果：治疗量+50%，被治疗者攻击+15%

**结局B：利用商心慈（黑暗路线）**
- 条件：帮助黑土 + 不安慰商心慈
- 结果：商心慈受伤离开，黑土给予报酬
- 奖励：获得黑土的"暗道传承"
- "暗道传承"效果：暗杀伤害+30%，隐身时间+50%
- 代价：商心慈关系线永久断裂

**结局C：救赎路线（中立→正义）**
- 条件：先旁观，后帮助商心慈
- 结果：商心慈受伤但最终恢复，黑土被驱逐
- 奖励：商心慈关系达到亲密，获得"商道传承"
- "商道传承"效果：交易价格-20%，元石收入+15%

## 4. 三王传承副本

### 4.1 副本概述

三王传承是南疆最古老的传承之一，对应Terraria的三个机械Boss。三个传承分别由三位远古蛊王留下，各有特色。

**副本入口**：
- 位置：南疆荒野深处
- 解锁条件：收集3张地图碎片（行商购买/任务奖励/荒兽掉落）
- 进入方式：在特定祭坛使用地图碎片

### 4.2 铁骨王传承（= Destroyer）

**铁骨王背景**：
远古骨道蛊王，以骨骼为蛊，统御亡灵大军。传承位于地下骨窟。

**副本设计**：

```
[入口：骨窟大门]
    │
    ▼
[第一层：骨海走廊]
    │   - 环境：白骨遍地，幽蓝磷火
    │   - 敌人：骷髅兵×20，骨弓手×5
    │   - 机关：骨刺陷阱（地面突然刺出骨刺）
    │   - 谜题：按骨骼大小排列5个骷髅头
    │
    ▼
[第二层：骨王殿堂]
    │   - 敌人：骨将军×2（小Boss，HP=5000）
    │   - 机关：骨牢（被骨手抓住3秒）
    │   - 谜题：在骨壁上找到正确的符文顺序
    │
    ▼
[Boss战：铁骨王]
```

## 5. 春秋蝉重生机制

### 5.1 获取春秋蝉

**获取途径**：
- 三王传承副本通关后，有5%概率在最终宝箱中获得
- 黑市商人偶尔出售（价格：2000元石）
- 完成特定隐藏任务线

### 5.2 重生机制

**触发条件**：
- 玩家死亡时背包中有春秋蝉
- 春秋蝉处于激活状态（右键使用后激活，持续1小时）

**重生流程**：

```
[玩家死亡]
    │
    ▼
[检测春秋蝉]
    │   ├─ 有且激活 → 弹出选择框
    │   └─ 无或未激活 → 正常死亡
    │
    ▼
[选择框]
    │   "春秋蝉感应到你的死亡。是否启动时光回溯？"
    │   ├─ "是" → 启动重生
    │   └─ "否" → 正常死亡
    │
    ▼
[时光回溯动画]
    │   ├─ 0-2秒：屏幕逐渐变为金色
    │   ├─ 2-4秒：时间倒流特效（粒子逆向运动）
    │   ├─ 4-6秒：世界恢复到死亡前状态
    │   └─ 6-8秒：玩家在安全点复活
    │
    ▼
[重生完成]
    ├─ 世界恢复到死亡前30秒的状态
    ├─ 玩家保留：经验值、蛊虫、装备
    ├─ 玩家失去：死亡时掉落的元石
    ├─ 春秋蝉消耗（消失）
    └─ 获得"时光回溯"Debuff：10分钟内不可再次使用春秋蝉
```

### 5.3 技术实现

**世界快照系统**：
- 每30秒自动保存一次世界快照
- 快照内容：Tile状态、NPC位置、敌人状态、物品状态
- 快照不保存：玩家位置（复活到安全点）、玩家背包
- 快照存储：内存中保留最近3个快照，超出覆盖

**数据结构**：

```csharp
public class ChunQiuChanSystem : ModSystem
{
    public class WorldSnapshot
    {
        public int Tick;
        public Tile[,] Tiles;       // 变化的Tile
        public List<NPCData> NPCs;  // NPC状态
        public List<ItemData> Items; // 掉落物品
    }

    private Queue<WorldSnapshot> _snapshots = new();
    private const int SnapshotInterval = 30 * 60; // 30秒(60fps)
    private const int MaxSnapshots = 3;

    public bool HasChunQiuChan(Player player);
    public bool IsChunQiuChanActive(Player player);
    public void RestoreSnapshot();
}
```

| 限制项 | 数值 | 说明 |
|--------|------|------|
| 冷却时间 | 10分钟 | 两次使用间隔 |
| 快照回溯 | 30秒 | 回溯到30秒前 |
| 每日使用次数 | 1次 | 每天只能使用1次 |
| Boss战中 | 不可用 | Boss战期间无法回溯 |
| PvP中 | 不可用 | PvP中无法回溯 |
