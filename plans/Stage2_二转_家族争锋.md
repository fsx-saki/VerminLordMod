## 1. 三寨大比事件链

### 1.1 事件触发条件

- StoryPhase >= FamilyRecognition（一转阶段完成）
- 玩家修为 >= 二转初阶
- 完成一转所有主线任务
- 游戏时间 >= 进入山寨后30天

### 1.2 资格赛（预选）

**流程图**：

```
[古月博宣布三寨大比]
    │
    ▼
[资格赛报名] ──── GuYueChief对话触发
    │               报名费：10元石
    │
    ▼
[古月内部选拔] ──── 连续3场战斗
    │    │
    │    ├─ 第1场：vs GuYueFirstTurnGuMaster(强化) 
    │    │         HP:1500 DMG:25 DEF:15
    │    │
    │    ├─ 第2场：vs GuYueSecondTurnGuMaster(普通)
    │    │         HP:3000 DMG:40 DEF:25
    │    │
    │    └─ 第3场：vs GuYueFistInstructor(强化)

    │

    ├─ 全胜：直接获得参赛资格 + "古月之星"称号

**资格赛战斗规则**：
- 战斗场地：古月山寨演武场（封闭区域，不可逃跑）
- 每场战斗间隔：可休息30秒（恢复生命/真元）
- 禁止使用：恢复类丹药（仅允许战斗中自然恢复）
- 胜利条件：对方生命归零或认输

## 2. 天鹤上人Boss战

### 2.1 触发条件

- 三寨大比完成后
- StoryPhase == Crisis
- 白凝冰事件触发后
- 玩家修为 >= 二转中阶

### 2.2 Boss背景

天鹤上人，五转蛊师，青茅山最强者之一。风道传承者，鹤形化身。在小说中是青茅山的守护者级人物，但同时也是利益争夺的参与者。

### 2.3 Boss规格

| 属性 | 数值 |
|------|------|
| 生命值 | 25000（普通）/ 37500（专家）/ 50000（大师） |
| 伤害 | 65 / 97 / 146 |
| 防御 | 35 |
| 击退抗性 | 80% |
| 对应Terraria Boss | Eye of Cthulhu / Eater of Worlds 级别 |

### 2.4 Phase 1：风刃攻击（100%-60%）

**行为模式**：
- 天鹤上人站在地面，以人形施法
- 移动方式：缓慢飘浮，偶尔瞬移

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 风刃斩 | 发射3道弧形风刃弹幕 | 55 | 2秒 |
| 龙卷风 | 召唤小型龙卷风追踪玩家 | 40/秒 | 8秒 |
| 风墙 | 生成风墙阻挡弹幕和移动 | 0 | 15秒 |
| 至亲血虫追踪 | 发射红色追踪弹（模拟至亲血虫） | 70 | 12秒 |

**至亲血虫追踪机制**：
- 弹幕颜色：暗红色，带血迹粒子
- 追踪方式：先直线飞行1秒，然后追踪玩家
- 追踪时间：5秒，之后直线飞出
- 可被攻击消灭（HP=50）
- 被击中时施加"血脉追踪"Debuff：5秒内被标记，下次风刃必中

### 2.5 Phase 2：鹤形空战（60%-25%）

**转换动画**：
- 天鹤上人仰天长啸："鹤鸣九皋！"
- 身体膨胀，变形为巨大白鹤
- 全屏风压效果，玩家被推开

**行为模式**：
- 以鹤形飞行，高速移动
- 俯冲攻击为主

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 鹤翼斩 | 俯冲时两侧发射风刃 | 60 | 3秒 |
| 鹤喙突刺 | 高速俯冲，直线贯穿 | 80 | 6秒 |
| 羽刃风暴 | 盘旋上升，散落大量羽刃弹幕 | 35/枚 | 10秒 |
| 风之领域 | 停留空中，周围生成持续风压区 | 15/秒 | 20秒 |

**鹤形移动模式**：
- 在玩家上方盘旋 → 俯冲攻击 → 拉升 → 再次盘旋
- 俯冲有预兆：头部闪光0.5秒后冲下
- 盘旋时是攻击窗口（防御降低50%）

### 2.6 Phase 3：存息玉葬蛊防御（25%-0%）

**转换动画**：
- 天鹤上人回归人形，面色苍白
- "存息玉葬蛊！"
- 周围生成半透明玉色护盾

**行为模式**：
- 防御大幅提升，但移动缓慢
- 攻击频率降低但伤害更高
- 玉葬蛊护盾需要先击破才能造成伤害

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 玉葬护盾 | 生成500HP护盾，期间免疫伤害 | — | 护盾破碎后15秒 |
| 风鹤合击 | 同时发射风刃+鹤形弹幕 | 90 | 5秒 |
| 存息一击 | 蓄力3秒后发射巨型风球 | 120 | 20秒 |
| 绝命风暴 | HP<10%时，全屏风刃风暴 | 50/秒 | 一次性 |

**护盾机制**：
- 护盾HP：500（普通）/ 750（专家）/ 1000（大师）
- 护盾存在时天鹤上人免疫伤害
- 护盾弱点：土道/金道蛊虫对护盾伤害×2
- 护盾破碎时天鹤上人眩晕2秒

### 2.7 掉落物

|------|------|------|

## 3. 白凝冰自爆事件

### 3.1 触发条件

- 天鹤上人Boss战完成后
- StoryPhase == Crisis
- 游戏时间：Boss战后1-3天自动触发

### 3.2 过场动画

**动画序列（约2分钟）**：

| 时间 | 画面 | 旁白/对话 |
|------|------|-----------|
| 0-15s | 白凝冰站在山顶，风雪环绕 | — |
| 15-30s | 白凝冰举起双臂，冰蓝色光芒聚集 | "北冥冰原……借我之力！" |
| 30-45s | 双臂开始碎裂，化为冰晶 | "我的双臂……就当是给你们的礼物吧！" |
| 45-60s | 巨大冰蓝色能量球向四周扩散 | 全屏白光 |
| 60-90s | 青茅山逐渐被冰雪覆盖 | "从此以后，青茅山……将永记白凝冰之名！" |

- 使用ScreenShader实现全屏白光

- 使用Tile替换实现冰雪覆盖

| 古月山寨 | 部分建筑损坏，NPC受冻 | 永久 |
| 全局 | 降雪天气 | 7天 |

**冰封期间新敌人**：

| 敌人 | 生命 | 伤害 | 说明 |

## 4. 古月一代Boss战

### 4.1 触发条件

- 白凝冰自爆事件后
- 铁血冷事件链推进到特定阶段
- 玩家修为 >= 二转高阶
- 隐藏Boss，需主动触发

### 4.2 Boss背景

古月一代，五转蛊师，古月家族的隐藏强者。血道修炼者，暗中操控家族事务。在小说中是方源利用血颅蛊提升资质的关键人物。

### 4.3 Boss规格

| 属性 | 数值 |
|------|------|
| 生命值 | 35000（普通）/ 52500（专家）/ 70000（大师） |
| 伤害 | 80 / 120 / 180 |
| 防御 | 45 |
| 击退抗性 | 85% |
| 对应Terraria Boss | Deerclops级别 |

### 4.4 Phase设计

**Phase 1：血幕天华（100%-70%）**

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 血幕天华 | 生成红色屏障，阻挡弹幕 | — | 20秒 |
| 血刃斩 | 发射弧形血刃弹幕 | 70 | 3秒 |
| 血液操控 | 地面涌出血液柱 | 55 | 6秒 |
| 血蛊寄生 | 发射追踪血蛊弹，命中后持续掉血 | 30/秒×5秒 | 15秒 |

**血幕天华机制**：
- 红色半透明屏障覆盖古月一代
- 屏障HP：800
- 弹幕被屏障阻挡
- 近战可穿透屏障（但伤害降低50%）
- 屏障破碎后古月一代进入短暂硬直

**Phase 2：血道觉醒（70%-35%）**

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 血海翻涌 | 大范围血液波浪 | 60 | 8秒 |
| 血影分身 | 生成2个血影分身（HP=1000） | 同本体 | 25秒 |
| 血脉共鸣 | 对被血蛊寄生的玩家造成额外伤害 | +50% | — |
| 吸血之术 | 近战攻击恢复自身HP | 50 | 10秒 |

**血影分身机制**：
- 分身外观与本体相同但颜色偏暗
- 分身会使用Phase 1的所有攻击（伤害×0.6）
- 分身被消灭时爆炸，造成范围伤害
- 真身脚下有微弱红色光环（辨识方法）

**Phase 3：血颅献祭（35%-0%）**

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 血颅蛊 | 召唤巨大血色骷髅头弹幕 | 100 | 8秒 |
| 血祭领域 | 全屏持续伤害（5/秒），玩家持续掉血 | 5/秒 | 持续 |
| 狂血 | 自身攻击力+50%，防御-30% | — | HP<20%触发 |
| 最终一击 | 蓄力5秒后释放全屏血爆 | 150 | HP<5%触发 |

### 4.5 掉落物

| 物品 | 掉率 | 说明 |
|------|------|------|
| 血颅蛊 | 100% | 资质提升物品（丙等→甲等） |
| 血道传承印记 | 80% | 解锁血道修炼路线 |
| 古月秘宝 | 50% | 家族隐藏宝物 |
| 血海雏形(蛊虫) | 15% | 范围攻击蛊虫 |
| 元石×200 | 100% | 基础奖励 |

**血颅蛊使用规则**：
- 使用后资质提升一个等级（丙→乙→甲）
- 使用时需要消耗100元石
- 使用时有30%概率失败（失败后血颅蛊不消失，但元石消耗）
- 十绝体不可使用
- 使用后玩家获得"血脉觉醒"Buff：24小时内攻击+20%

## 5. 铁血冷事件链

### 5.1 事件流程

```
[铁血冷抵达古月山寨]
    │   触发：贾金生之死事件后，冰封事件前
    │
    ▼
[调查阶段1：收集情报]
    │   ├─ 与各NPC对话获取线索
    │   ├─ SearchSystem搜索现场
    │   └─ 铁血冷逐步缩小嫌疑人范围
    │
    ▼
[调查阶段2：锁定古月一代]
    │   ├─ 发现古月一代与多起失踪案有关
    │   ├─ 铁血冷决定正面调查古月一代
    │   └─ 玩家可选择协助或阻止
    │
    ▼
[对峙：铁血冷 vs 古月一代]

## 8. 地脉守护者Boss（Wall of Flesh等效）

### 8.1 触发条件

- 离开青茅山后
- 玩家修为 >= 二转巅峰
- 在地下深处找到地脉节点
- 将"古月推荐信"投入地脉裂缝

### 8.2 Boss设计理念

地脉守护者是青茅山地脉的守护灵，连接着地下所有矿脉节点。击败它将打破地脉封印，释放更深层的精神矿脉，触发Hardmode转换。

### 8.3 Boss规格

| 属性 | 数值 |
|------|------|
| 生命值 | 40000（普通）/ 60000（专家）/ 80000（大师） |
| 伤害 | 70 / 105 / 158 |
| 防御 | 30 |
| 击退抗性 | 100% |
| 类型 | 不移动型Boss（类似Wall of Flesh） |

### 8.4 战斗机制

**地脉连接**：
- Boss连接着3-5个地脉矿石节点
- 每个节点为Boss提供：生命恢复50/秒 + 防御+10
- 节点可被攻击摧毁（HP=2000/个）
- 摧毁节点时Boss短暂眩晕

**攻击模式**：

| 攻击名 | 描述 | 伤害 | 冷却 |
|--------|------|------|------|
| 地脉冲击波 | 前方扇形地面冲击 | 60 | 4秒 |
| 矿脉弹射 | 从节点发射矿石弹幕 | 45 | 3秒 |
| 地震 | 全屏震动，地面生成裂缝 | 30 | 10秒 |
| 地脉吞噬 | 将玩家拉向Boss | 0 | 15秒 |
| 矿晶风暴 | 所有节点同时发射弹幕 | 35/枚 | 20秒 |

**Phase转换**：

| Phase | 触发 | 变化 |
|-------|------|------|
| Phase 1 | 100%-50% | 3个节点，基础攻击 |
| Phase 2 | 50%-20% | 5个节点（新生成2个），攻击频率+50% |
| Phase 3 | 20%-0% | 节点全部爆炸，Boss防御-50%，攻击+100% |

### 8.5 击败后世界变化

**Hardmode转换事件**：

1. 地脉守护者死亡动画：身体碎裂，释放大量灵气
2. 全屏白光2秒
3. 世界生成新矿脉：
   - 灵晶矿（替代部分铜/铁矿）
   - 赤金矿（替代部分银/钨矿）
   - 天晶矿（新Hardmode矿石）
4. 地下深处生成新的敌人生成点
5. 南疆荒野Biome正式激活
6. 玩家获得"破界者"成就
7. StoryPhase → Hardmode

### 8.6 掉落物

| 物品 | 掉率 | 说明 |
|------|------|------|
| 地脉之心 | 100% | Hardmode核心材料 |
| 灵晶矿石×30 | 100% | 新矿石 |
| 赤金矿石×20 | 100% | 新矿石 |
| 地脉守护者之魂 | 50% | 制作强力装备 |
| 破界锤 | 30% | 可破坏Altar，生成新矿 |
