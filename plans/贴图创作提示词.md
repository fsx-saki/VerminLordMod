# 贴图创作提示词

> 以下为 VerminLordMod 中所有待创作贴图的详细提示词。
> 每个条目包含：文件名、路径、尺寸规格、帧结构说明、风格参考、以及 AI 绘图提示词（中英双语）。
> 所有贴图为 **tModLoader 像素风格**，16×16 像素为基本单元格，适配 Terraria 的 2D 像素艺术风格。

## 通用规范

| 项目 | 规范 |
|------|------|
| 风格 | Terraria 像素风 (Pixel Art)，16×16 基础单元格 |
| 调色板 | 每像素 8-bit RGBA，透明背景 |
| 单元格尺寸 | 16×16 px（底部行有时为 18px，含 2px 内边距） |
| 多帧布局 | 水平排列（StyleHorizontal=true）或垂直排列 |
| 文件格式 | PNG，非交错 (non-interlaced) |
| 命名规则 | `ClassName.png`，特殊用途加 `_Flame` 等后缀 |
